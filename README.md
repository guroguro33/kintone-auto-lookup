# kintone プラグイン　ルックアップ先自動更新プラグイン

## ディレクトリ・ファイル構成

#### config

- 設定画面用

#### desktop

- カスタマイズビュー用

#### static

- プラグイン生成用
- config や desktop のビルド先

#### types

- 型定義

#### vite.config.ts

- 設定画面用 vite 設定

#### vite.plugin.config.ts

- カスタマイズビュー用 vite 設定

## 設計

- まずは自フィールドのルックアップフィールドは１つ
  - ルックアップ元フィールド
- ルックアップ先のフィールドは以下を選択する
  - アプリ ID
  - 更新したいルックアップフィールドコード（≒ ルックアップ元フィールド）
